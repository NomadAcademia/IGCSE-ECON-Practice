# 📈 IGCSE Economics Master Pro - Vocabulary Trainer

A gamified, interactive vocabulary tool designed for IGCSE Economics students. This tool uses active recall and self-assessment to help students master complex economic definitions.

## 🚀 How to use (For Students)

### 1. Open the Link
Open the link provided in your browser (QQ Browser, WeChat, or Edge).

### 2. Add to Home Screen (CRITICAL)
To use this like a real app and save your progress (XP & Streaks), do the following:
* **On iPhone (Safari):** Tap the **Share** button (square with arrow) -> **Add to Home Screen**.
* **On Android (Edge):** Tap the **three dots (...)** -> **Add to phone**.
* **On QQ Browser:** Tap the **Menu (three lines)** -> **Toolbox** -> **Add Shortcut**.

### 3. How to Play
1.  Read the **Vocabulary Word** at the top.
2.  Type your definition in the box.
3.  Click **Check Logic** to see the official answer.
4.  Compare your answer with the official one. 
5.  If you got the core concept right, click **"Nailed It!"** to earn XP and build your fire streak 🔥.

---

## 🛠 Features for Teachers
* **Gamified XP System:** Students earn higher ranks (e.g., "Central Bank Governor") as they practice.
* **ESL Friendly:** Provides a side-by-side comparison of the student's answer and the official definition.
* **Responsive Design:** Works on all screen sizes (Phone, Tablet, PC).
* **Offline Ready:** Once added to the home screen, it can be used even with a weak connection.

## 📝 Editing the Vocabulary
To update the word list:
1.  Open `index.html`.
2.  Find the `const vocabData = [...]` section.
3.  Add or edit words using this format:
    `{word: "Inflation", def: "A sustained rise in the general price level"},`
4.  Save and commit changes; the website will update automatically!
